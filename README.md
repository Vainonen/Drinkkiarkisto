Drinkkiarkisto
==============

1. Johdanto
Työaihe: drinkkiarkisto
Ohjelmoitikieli: PHP
Tietokanta: PostgreSQL
Drinkkiarkiston on tarkoitus toimia erilaisten drinkkireseptien hakuun, joko hakusanalla tai
selaamalla drinkkejä aakkosjärjestyksessä. Tietokanta näyttää yksittäisen drinkin lajityypin,
ainesosat ja vaihtoehtoiset nimitykset. Järjestelmään voi myös ehdottaa omia tai puuttuvia
reseptejä, jotka järjestelmän ylläpitäjä hyväksyy tai antaa käyttäjälle suoraan oikeudet
viedä resepti tietokantaan.
Tietokanta on tarkoitus toteuttaa laitoksen users-palvelimella users/okvainon/ht docskansiossa.
Web-sovellus on toteutettu PHP-kielellä ja tarkoitus on käyttää laitoksen
PostgreSQL-tietokantapalvelinta.
